# PulsIQ 🧠💙

PulsIQ is an AI-powered health intelligence platform designed to simplify complex medical data into clear, actionable insights.

## 🚀 Features
- AI Health Mentor (Pulsy)
- Prescription Scanner
- Symptom Analysis
- Health Data Tracking
- Smart Recommendations

## 🎯 Vision
To bridge the gap between medical complexity and human understanding using AI.

## 🛠 Tech Stack
- HTML, CSS, JavaScript (initial)
- AI Integration (future)
